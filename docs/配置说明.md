# Docusaurus 文档站点配置说明

本文档说明如何添加新栏目、排序、隐藏/显示设置。

## 目录结构

```
docs/
├── mac/
│   ├── _category_.json      # 栏目配置
│   ├── index.md            # 栏目首页
│   ├── 目录111/
│   │   ├── _category_.json
│   │   └── *.md            # 文档文件
│   └── 目录222/
│       ├── _category_.json
│       └── *.md
└── ubuntu/
    └── ...
```

## 一、添加新栏目

### 步骤1：创建栏目文件夹

在 `docs/` 目录下创建新栏目文件夹，例如 `docs/tool/`

### 步骤2：创建 _category_.json

在栏目文件夹中创建 `docs/tool/_category_.json`：

```json
{
  "label": "工具",
  "position": 4,
  "link": null
}
```

- `label`: 栏目显示名称
- `position`: 在左侧侧边栏中的顺序（数字越小越靠前）
- `link`: 设为 `null` 隐藏栏目标题，或设置 `generated-index` 显示目录列表页

### 步骤3：创建文档文件

在栏目文件夹中创建 `docs/tool/index.md` 作为栏目首页：

```yaml
---
title: 工具
---

# 工具

欢迎来到工具页面。
```

### 步骤4：创建子目录（可选）

在栏目下创建子目录，如 `docs/tool/目录1/`、`docs/tool/目录2/`

在子目录中创建 `_category_.json`：

```json
{
  "label": "目录1",
  "position": 1,
  "link": {
    "type": "generated-index",
    "title": "目录1",
    "description": "目录1下的文档"
  }
}
```

在子目录中创建 md 文档文件。

### 步骤5：配置顶部导航

编辑 `docusaurus.config.js`，在 `navbar.items` 中添加：

```javascript
{ type: "docSidebar", sidebarId: "toolSidebar", position: "left", label: "工具" },
```

### 步骤6：配置左侧侧边栏

编辑 `sidebars.js`，添加新的侧边栏配置：

```javascript
toolSidebar: [
  {
    type: 'category',
    label: '目录1',
    collapsed: false,
    link: { type: 'generated-index', title: '目录1', description: '目录1下的文档' },
    items: [
      { type: 'doc', id: 'tool/目录1/doc1' },
      { type: 'doc', id: 'tool/目录1/doc2' },
    ],
  },
  {
    type: 'category',
    label: '目录2',
    collapsed: false,
    link: { type: 'generated-index', title: '目录2', description: '目录2下的文档' },
    items: [
      { type: 'doc', id: 'tool/目录2/doc1' },
    ],
  },
],
```

## 二、栏目排序

### 方法1：修改 _category_.json

在栏目的 `_category_.json` 中修改 `position` 值：

```json
{
  "label": "工具",
  "position": 5
}
```

数字越小越靠前。

### 方法2：修改 sidebars.js

在 `sidebars.js` 中调整数组顺序：

```javascript
macSidebar: [
  // 排在前面的目录
  { type: 'category', label: '目录222', ... },
  { type: 'category', label: '目录111', ... },
],
```

### 方法3：修改 docusaurus.config.js

调整顶部导航栏中的顺序：

```javascript
navbar: {
  items: [
    // 调整这里的位置
    { type: "docSidebar", sidebarId: "macSidebar", position: "left", label: "mac" },
    { type: "docSidebar", sidebarId: "toolSidebar", position: "left", label: "工具" },
  ]
}
```

## 三、隐藏/显示设置

### 隐藏栏目标题

在栏目的 `_category_.json` 中设置：

```json
{
  "label": "工具",
  "link": null
}
```

### 隐藏某个文档

在 md 文件的 frontmatter 中添加：

```yaml
---
sidebar_label: false
---
```

### 隐藏某个子目录

从 `sidebars.js` 的 `items` 数组中移除对应项，或在子目录的 `_category_.json` 中设置：

```json
{
  "label": "目录1",
  "link": null
}
```

### 隐藏顶部导航入口

在 `docusaurus.config.js` 中注释或删除对应行：

```javascript
// 注释掉这一行
// { type: "docSidebar", sidebarId: "toolSidebar", position: "left", label: "工具" },
```

### 展开/折叠目录

在 `sidebars.js` 的 category 中设置 `collapsed`：

```javascript
{
  type: 'category',
  label: '目录1',
  collapsed: false,  // false = 展开，true = 折叠
  items: [...]
}
```

## 四、常用配置示例

### 完整栏目配置示例

```javascript
// sidebars.js
ubuntuSidebar: [
  {
    type: 'category',
    label: '目录2',           // 左侧显示的名称
    collapsed: false,        // 是否折叠（false=展开）
    link: {
      type: 'generated-index',
      title: '目录2',         // 生成的索引页标题
      description: '目录2下的文档列表'
    },
    items: [
      { type: 'doc', id: 'ubuntu/目录2/cmd' },
      { type: 'doc', id: 'ubuntu/目录2/intro' },
    ],
  },
],
```

### 完整 _category_.json 示例

```json
{
  "label": "目录名称",
  "position": 1,
  "link": {
    "type": "generated-index",
    "title": "目录标题",
    "description": "目录描述"
  }
}
```

## 五、验证配置

修改完成后，运行以下命令验证：

```bash
# 清除缓存并构建
npm run clear && npm run build

# 或启动开发服务器
npm start
```

访问对应页面检查侧边栏是否正确显示。
